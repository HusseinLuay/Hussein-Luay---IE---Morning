    1 - serief : 
Fonts with small decorative strokes at the end of characters.

    2 - Sans-Serif: 
 Fonts without the decorative strokes at the end of characters.

    3 - Monospace: 
 Fonts where each character takes up the same horizontal space.

    4 - Cursive: 
 Fonts that mimic handwriting or have a script-like appearance.

    5 - Fantasy: 
 Fonts with decorative and artistic styles that don't fit into the other categories.

2.It is recommended to use each family of generic fonts in fallback style. Why?

because not all users will have the same fonts installed on their devices. If a specific font is not available,
the browser will move on to the next font in the list. Providing fallbacks ensures that there is always a defined font 
for the text, even if the preferred font is not present.
