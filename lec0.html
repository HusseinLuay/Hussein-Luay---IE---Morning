1. What is the web development and what are types of it?

Web Programming (or web development) is the art of constructing and maintaining web pages for websites by a web developer (also known as a web engineer) to give the user what he needs from the website.

the types of web development :

  1. Frontend
  2. Backend
  3. Fullstack

2. What is difference between web developers and web designers

web developer focus on the technical aspects of building a website, including coding, server management, and database management. and he is Proficient in programming languages like HTML, CSS, JavaScript, and backend languages (e.g., Python, Ruby, PHP) and he handle the functionality and interactivity of a site, ensuring it works smoothly and efficiently

web designer focus is on the visual aspects and user experience of a website. so he Proficient in design tools (like: Adobe XD, Sketch , figma ) and have an understanding of UI/UX principles. he create layouts, choose color schemes, design graphics, and ensure a site's aesthetics align with the brand or purpose.
